5. Error: java: illegal start of type
   Error: java: ';' expected
   ...

   Error: java: class,interface, or enum expected
   ...

    Язык Java не поддерживает множественное наследование реализаций (классов).
    Это объясняется тем, что такое наследование порождает некоторые проблемы.

    При множественном наследовании вызов метода родительского класса, привел бы к неопределенности,
    так как компилятор не знает о том, метод какого именно суперкласса должен быть вызван.

    Вместо множественного наследования классов в Java введено множественное наследование интерфейсов,
    при котором, как утверждается, никаких проблем не возникает.
